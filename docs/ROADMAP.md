# 🚀 Asset Portfolio Manager

> Personal desktop application for managing digital assets and multiple wallets.
>
> **Status:** 🚧 In Development
>
> **Current Version:** v0.10

---

# Vision

Asset Portfolio Manager is an offline-first desktop application designed to manage assets distributed across multiple wallets.

The application focuses on:

- Portfolio management
- Wallet management
- Transaction history
- Daily and weekly goals
- Portfolio analytics
- Local-first storage
